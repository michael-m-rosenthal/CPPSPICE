44 -Particulars
45 
46  This header file defines interface macros to be called in place of
47  CSPICE user-interface-level functions. Currently, the sole purpose
48  of these macros is to implement automatic type casting under some
49  environments that generate compile-time warnings without the casts.
50  The typical case that causes a problem is a function argument list
51  containing an input formal argument of type
52 
53  const double [3][3]
54 
55  Under some compilers, a non-const actual argument supplied in a call
56  to such a function will generate a spurious warning due to the
57  "mismatched" type. These macros generate type casts that will
58  make such compilers happy.
59 
60  Examples of compilers that generate warnings of this type are
61 
62  gcc version 2.2.2, hosted on NeXT workstations running
63  NeXTStep 3.3
64 
65  Sun C compiler, version 4.2, running under Solaris.
